Service Availability

This tab provides an overview for the availability of a service. The API Engine can expose multiple maketplaces, and a service can be exposed on one or more API Marketplaces. Exposure doesn't mean that the API will show up in the available API's. You have to explicitly enable 'Show' in order to do so. When an API is exposed but not visible, an IP Restriction policy will be added with default CIDR/IP black and white listing.

The available API Marketplaces for the current service are shown beneath. The 'Show' or 'Hide' denotes the visibility of the API on the API Marketplace. Remember that when an API is available, but hidden, the default IP Restriction policy is applied.
